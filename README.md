# Student Service API <img align="right" src="https://img.shields.io/badge/Flask-000000?style=for-the-badge&logo=flask&logoColor=white" alt="Flask logo"/>
An API to control student information for an educational institution.  
The tecnologies used on this project are Python, Flask (web framework), Swagger and SQL Alchemy. The database is an SQLite file, generated when you run the application for the first time.  
  
*This project was created as an assessment of my skills with these tecnologies.*
## :scroll: Get started
### Prerequisites
- [Python3](https://www.python.org/downloads/)
### Install python dependencies
```$ python -m pip install -r requirements.txt ```

### :running: Run the application
Inside the project folder on command prompt, text:  
```$ python -m flask run --no-debugger```

### :telescope: Open your browser
The application will be running on port 5000, so access  
```http://localhost:5000```

## :gear: How does it work
The page is generated by Swagger.  
There will be some tabs guiding you to execute operations on the system. Generally, the API is rest and contains CRUD operations.

- Course
